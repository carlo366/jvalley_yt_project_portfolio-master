# Portfolio Saya dengan React, Vitejs dan Tailwind CSS

```bash
git clone https://github.com/jemblonganvalley/jvalley_yt_project_portfolio
cd jvalley_yt_project_portfolio
pnpm install
pnpm run dev 
```
